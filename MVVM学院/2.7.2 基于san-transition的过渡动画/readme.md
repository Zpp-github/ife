#### 任务目的
- 学习并实践CSS过渡动画的使用方法
- 学习并实践在San中使用san-transition开发过渡动画

#### 任务描述
实现一个简单的条目列表。当用户点击其中一个条目的时候，该条目会有简单的transition变换效果，效果结束后，弹层展示用户点击的条目详情。

#### 注意事项
- 你可以自己设计题目中的transition变换效果
- 请尝试使用san-transition实现这一效果
- 结合使用san-transition的动画生命周期钩子来实现这一效果
